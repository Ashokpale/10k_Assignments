Tables

  • It is used to show data in tabular format in a document.
  • Tables have rows ans cells.
  • Tables are defined with the table tag.
  • Table rows are defined with "tr"-tag
  • Table cells are defined with "td"-tag
  • Table heading are defined with "th"-tag

Comments

comments are used to insert notes or annotations within the code that are ignored by the web browser when rendering the webpage. They are helpful for documenting the purpose or functionality of certain sections of HTML code, making it easier for developers to understand and maintain the codebase. It is also used for preventing the code execution
